# GloFAS Seed-Repeat Application Batch, 2026-05-26

This note documents the second-seed application batch used to assess reservoir
seed sensitivity for the active GloFAS latent-path Q-DESN candidates.

## Purpose

The batch repeats the 16 active application candidates with the same model
specifications, inference settings, priors, input contract, and launch protocol,
but changes the reservoir seed from `20260512` to `20260526`. The raw GloFAS
baseline rows are unchanged. The Q-DESN rows receive seed-specific identifiers
and model-grid copies so that scores and figures can be compared without
overwriting the first-seed runs.

## Prepared Files

- Config/model-grid preparation script:
  `application/scripts/03_prepare_seed_repeat_batch_20260526.R`
- Launch script:
  `application/scripts/03_launch_seed_repeat_batch_20260526.sh`
- Launch manifest generated by the preparation script:
  `application/config/glofas_seedrepeat16_20260526_seed20260526_launch_manifest.csv`

Each copied YAML config uses a seed-specific application name, a seed-specific
cache directory, and a model-grid copy whose Q-DESN row has
`reservoir_seed = 20260526`.

## Reproducible Preparation

```bash
Rscript application/scripts/03_prepare_seed_repeat_batch_20260526.R \
  --seed 20260526 \
  --batch_id seedrepeat16_20260526_seed20260526 \
  --core_start 32
```

## Reproducible Launch

```bash
bash application/scripts/03_launch_seed_repeat_batch_20260526.sh \
  application/config/glofas_seedrepeat16_20260526_seed20260526_launch_manifest.csv
```

The launch script creates tmux session
`glofas_seedrepeat16_parallel_20260526`, pins the 16 runs to cores `32--47`,
and sets BLAS/OpenMP thread counts to one for each run.

## Interpretation Contract

This is a seed-sensitivity batch, not a new model-family search. Compare each
second-seed Q-DESN score, diagnostics, and figures to the corresponding
first-seed candidate before promoting any result. Large score or fit-shape
changes indicate reservoir-seed instability and should be treated as model
selection evidence rather than ignored noise.
